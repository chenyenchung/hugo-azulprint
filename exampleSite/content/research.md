interests:
  - tag: "chromosomes"
    label: "Chromosomes as dynamic structures"
    body: "A chromosome is not a static list of loci. With careful staining of pachytene nuclei, you can read its architecture directly — breaks, knobs, translocations — and watch that architecture change with the organism's history."
  - tag: "transposable-elements"
    label: "Controlling elements in maize"
    body: "Ac and Ds are the cleanest example I have of a locus whose state — silent, active, mutable — is governed by another element entirely. I'm tracing how the two-component system distributes itself across the genome, and what it does to nearby gene expression."
  - tag: "genome-response"
    label: "Genome response to challenge"
    body: "Reorganizations that look like noise at first — broken chromosomes rejoining, elements transposing in concert — often appear when the genome is stressed. I'd like to understand whether the response is a regulated program or the side-effect of one."
  - tag: "method"
    label: "A feeling for the organism"
    body: "I don't start from a model. I start from a kernel — sometimes from a single kernel — and let its variegation tell me which crosses to set up next. The method is patience, and counting."
